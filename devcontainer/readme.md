# Dev container

- arch linux for up to date repos (e.g. newest nvim)
  - image with AUR access and pavu
- auto-setup my dotfiles
  - tmux
  - nvim w LSPs
  - keep all installations local
- mount the folder it is run from
- [ ] Compilers as necessary

## Usage

`build.sh` to build the latest version
`run.sh` from wherever we want to edit
